# chemmaster-quiz
Short quiz for Secondary School Students taking O-level Chemistry 6092 in Singapore

This is a mini browser project by me to aid secondary school students in their O-level Chemistry revision. 

It is a gamified quiz platform and covers the topics of Experimental Chemistry, Kinetic Particle Theory, Atomic Structure, Chemical Bonding & Structure, Molar Calculations, Acid-Base Chemistry, Electrochemistry, Redox reactions, Rate of Reaction, Chemical Energetics, and Periodic Table.

To try out the quiz, go to the link https://jdtoh-max.github.io/chemmaster-quiz/ and it will run on your own device browser.
